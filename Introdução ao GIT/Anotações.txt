Navegar dentro das pastas: CD Linux e Windows e o nome da pasta para voltar as pastas CD .. 
Limpar a tela: CLS Windows clear Linux ctrl + L limpa no Linux
Criar pastas: MKDIR nome da pasta Linux e Windows 
Para criar pastas basta digitar echo oque vai conter dentro do arquivo e > nome do arquivo . tipo do arquivo exemplo (Echo hello > hello.txt ).
Del nome: server para deletar arquivos e não diretórios Windows
RMDIR /s /q: serve para excluir repositórios como pastas e não arquivos
